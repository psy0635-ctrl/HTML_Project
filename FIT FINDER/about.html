ABOUT PROJECT

Fit is not size.
Fit is balance.

FIT FINDER는 사용자가 자신의 체형에 맞는 옷의 실루엣을 쉽게 이해하고, 추천 핏과 피해야 할 핏을 한눈에 확인할 수 있도록 만든 체형별 패션 가이드 웹사이트입니다.
View Guide → GitHub →
01

Project Goal

체형별로 어울리는 실루엣과 피해야 할 핏을 안내하여, 사용자가 온라인 쇼핑과 코디 선택에서 실패를 줄일 수 있도록 돕는 것이 목표입니다.

02

Why This Topic?

옷을 살 때 “왜 나에게는 안 어울릴까?”라는 고민에서 시작했습니다. FIT FINDER는 체형을 숨기기보다, 나에게 맞는 균형을 찾도록 돕습니다.

03

Main Features

체형별 Guide, 실전 Style Tips, 추천/비추천 핏 정보를 제공합니다. Tips 페이지에서 Guide 페이지의 체형 카드로 바로 이동할 수 있습니다.

DESIGN POINT

Dark Glassmorphism UI

전체 디자인은 다크 톤과 글래스모피즘 효과를 활용하여 패션 아카이브처럼 고급스럽고 감각적인 분위기를 만들었습니다.
Dark Tone Glass Card Fashion Mood Hover Interaction Grid Layout

TECH STACK

HTML5 + CSS3

현재 버전은 JavaScript 없이 HTML5와 CSS3만 사용한 정적 웹사이트입니다. CSS Grid, Flexbox, Border, Hover, Transition, Glassmorphism 효과를 활용했습니다.
