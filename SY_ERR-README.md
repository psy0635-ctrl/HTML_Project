## 📌 1. 프로젝트 개요 (Overview)

| 항목 | 내용 |
|------|------|

## 🧠 3. 핵심 개념 (용어 설명 포함)

### 📦 데이터화 (Data Modeling)
👉 **데이터화란?**  
→ 실제 사물(옷)을 컴퓨터가 이해할 수 있도록 정보로 바꾸는 과정

예:
- 자켓 → 카테고리: 상의  
- 검정색 → 색상: Black  

---

### 🔎 필터링 (Filtering)
👉 **필터링이란?**  
→ 특정 조건에 맞는 데이터만 골라내는 기능

예:
- “겨울 옷만 보여줘”
- “스트릿 스타일만 보여줘”

---

### 🤖 추천 시스템 (Recommendation System)
👉 **추천 시스템이란?**  
→ 저장된 데이터를 기반으로 가장 적절한 결과를 자동으로 제안하는 기능

예:
- 자켓 + 바지 + 신발 조합 생성

---

### ⚡ SPA (Single Page Application)
👉 **SPA란?**  
→ 페이지를 새로고침하지 않고 하나의 페이지에서 내용만 바뀌는 웹 구조

✔ 장점:
- 빠른 속도
- 부드러운 사용자 경험

---
